Estos cuadernos contienen el código para las visualizaciones de la gestión en reservas y pedidos en Bodega. 
Se requiere tener instalado el paquete de python streamlit para ejecutar los cuadernos.
Para ejecutar los cuadernos se debe ejecutar el comando:
streamlit run nombreDelCuaderno.ipynb

El archivo racingBar2025PlanillaPCA.ipynb tiene una visualización del top de responsables de creación de las reservas.
El arcuivo racingBar2025PedidosUsers.ipynb tiene una visualización del top de usuarios que solicitaron productos en Bodega
El archivo racingBar2025PedidosProducts.ipynb tiene una visualización del top de productos que se solicitaron en Bodega
El archivo racingBar2025PedidosProducts.ipynb tiene una variación dado que se corrió en local y no en la nube de google colab.
